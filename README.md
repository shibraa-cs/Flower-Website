# Eternal Flowers - E-Commerce Flower Shop Website

A fully functional e-commerce flower shop website with shopping cart, user authentication, order history, and admin dashboard.

## 🌸 Live Demo
[View Live Website](https://shibraa-cs.github.io/Flower-Website/)

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- localStorage (for data persistence)
- Google Fonts

## ✨ Features

### For Regular Users
- Browse products by category
- Add items to shopping cart
- Update quantities in cart
- Checkout with order summary
- User registration and login
- View order history
- Persistent cart data

### For Admin Users
- Full admin dashboard
- View all users
- View all orders
- Revenue analytics
- User management
